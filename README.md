# Project Title

LLM Monitoring Dashboard.

## Description

This project exhibits an LLM monitoring dashboard where the user can customize the data visualization.

## Getting Started

### Dependencies

- The project's setup is based on Viet React TypeScript.

- Some dependencies like zustand for state management, tailwindcss for styling, tanstack query for data fetching, react grid layout for customizing grid items, recharts for data visualization.


### Installing

- Scaffolding is using the following command.
```
npm create vite@latest 
```

```
Project name : customizable-dashboard
Select a framework : React
Select a variant : Typescript/ React Compiler

```

- The other dependencies are installed by the below commands.

```
npm i zustand @tanstack/react-query

npm i  react-icons react-grid-layout recharts tailwindcss @tailwindcss/vite
```



### Executing program

- npm is utilized by the project to run locally

```
npm run dev
```
